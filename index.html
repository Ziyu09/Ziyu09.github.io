字符编码

所有的数据在计算机中的储存都是以二进制的形式储存的 文字也不例外
编码-将字符转换为二进制的过程
解码-将二进制转换为字符的过程
编码和解码所采用的规则称为字符集
国际通用字符集 UTF-8

实体

在 HTML 中有些时候我们不能直接书写一些特殊符号
比如：多个连需的空格 字母两侧的大于号和小于号
如果需要这些字符 则需要 HTML 文件中的转义字符（实体）
例：
&npsp; 空格
> 大于号
< 小于号
&copy; 版权符号

meta 标签

meta 标签主要用于设置网页中的一些元数据，元数据不是给用户看的
charset 指定网页的字符集
name 指定的数据的名称
content 指定数据的内容

keywords 表示网站的关键字 可同时使用多个关键字 不同关键字间用逗号隔开
description 用于指定网站的描述 会显示在搜索引擎的搜索的结果中
title 标签的内容会作为搜索结果的超链接文字

超链接

超链接可以让我们从一个页面跳转到其他页面
或者是当前页面的其他页面
超链接 行元素
在 a 标签中什么都能放

超链接的其他用法

target 属性 用来指定超链接的打开位置
可选值：

-self 默认值 在当前页面打开超链接
-blank 在一个新的页面中打开超链接
herf=”#” 回到顶部
herf=””
id 属性（唯一不重复的） -每一个标签都可以添加一个 id 属性
-id 属性是元素的唯一标识 同一个页面中不能出现重复的 id 属性 -可以跳转到页面的指定位置 只需要将 href 属性设置#目标元素的属性值
这是一个空白超链接

图片标签

用于引入一个外部的图片
使用 img 标签来引入外部图片，img 标签是一个自结束标签
属性：
src 属性指定是外部图片的路径 (路径规则和超链接是一样的)


alt 对图片的一个描述 这个描述默认不会显示 有些浏览器会在图片无法加载时显示 搜索引擎会根据 alt 中的内容来识别图片
width 图片的宽度 单位是像素
height 图片的高度 -宽度和高度如果只修改了一个 则另一个会等比例缩放
图片的格式：
jpg：支持的颜色丰富 不支持透明效果 不支持动图 一般来显示照片
gif：支持的颜色比较少 支持动图
png: 颜色丰富 支持复杂图片 专为网页而生
webp: 谷歌推出的 完美的图片格式 兼容性差

base64 格式：将图片使用 base64 进行编码 这样可以将图片转换为字符 一般都是需要个网页一起加载的图片才会使用
